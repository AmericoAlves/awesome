26. [StaDynA](https://github.com/zyrikby/StaDynA)   - 在存在动态代码更新功能（动态类加载和反射）的情况下支持安全应用程序分析的系统.  此工具结合了Android应用程序的静态和动态分析，以显示隐藏/更新的行为并使用此信息扩展静态分析结果.
