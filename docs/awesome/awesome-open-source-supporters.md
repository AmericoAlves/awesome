## Tags

### `requires-approval`

 虽然许多公司使用GitHub，Bitbucket和GitLab的公共存储库功能来确定项目的开源状态，但还有其他公司需要更多的验证.  这些公司已被标记为“requires-approval”标签，表明需要申请流程才能利用其开源优势.

### `non-commercial`

 一些公司将其产品用于开源，但仅用于非商业项目.  这些公司已被标记为“非商业”标签.
