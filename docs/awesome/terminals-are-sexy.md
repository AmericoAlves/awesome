## Shells

*性感外壳的性感框架，插件和资源.*

### Base Shells

* [Ammonite-Shell](https://github.com/lihaoyi/ammonite) - 具有类似Scala脚本语言的对象shell.
* [bash](https://www.gnu.org/software/bash/) -  GNU Project的shell（Bourne Again SHell）.
* [fish](https://fishshell.com/) - 智能且用户友好的命令行shell.
* [PowerShell](https://github.com/PowerShell/PowerShell) - 针对结构化数据进行优化的自动化和配置工具/框架.
* [xonsh](https://xon.sh/) -  Python-ish，BASHwards外观的shell语言和命令提示符.
* [zsh](https://www.zsh.org/) - 强大的shell脚本语言.

### ZSH

* [alf](https://github.com/psyrendust/alf) - 出于这个世界超快速和可配置的zsh框架.
* [ant-zsh](https://github.com/anthraxx/ant-zsh) - 小巧轻便的ZSH配置环境，可满足特殊定制需求.
* [antibody](https://github.com/getantibody/antibody) - 用Golang写的更快更简单的抗原.
* [antigen](https://github.com/zsh-users/antigen) -  zsh的插件管理器，受到了oh-my-zsh和vundle的启发.
* [dotzsh](https://github.com/dotphiles/dotzsh) -  zsh的社区驱动框架.
* [liquidprompt](https://github.com/nojhan/liquidprompt) - 针对Bash和Zsh的全功能和精心设计的自适应提示.
* [oh-my-zsh](https://github.com/robbyrussell/oh-my-zsh) - 用于管理zsh配置的令人愉快的社区驱动框架.
* [powerlevel9k](https://github.com/bhilburn/powerlevel9k) - 超级灵活的超棒电力线ZSH主题.
* [prezto](https://github.com/sorin-ionescu/prezto) -  Zsh的配置框架.
* [pumice](https://github.com/ryutamaki/pumice) -  zsh的轻量级插件管理器.
* [pure](https://github.com/sindresorhus/pure) - 漂亮，简约，快速的ZSH提示.
* [slimzsh](https://github.com/changs/slimzsh) -  ZSH的小型可用配置.
* [zeesh](https://github.com/zeekay/zeesh) - 跨平台Zsh框架.
* [zgen](https://github.com/tarjoilija/zgen) -  ZSH的轻量级简单插件管理器.
* [zilsh](https://github.com/zilsh/zilsh) -  Zsh配置系统旨在吸引更多的电力用户，并遵循vim-pathogen的简单方法.
* [zim](https://github.com/Eriner/zim) -  Zsh配置框架，具有超快的速度和模块化扩展.
* [zoppo](https://github.com/zoppo/zoppo) -  Zsh的残缺配置框架.
* [zplug](https://github.com/zplug/zplug) -  zsh的下一代插件管理器.
* [zpm](https://github.com/zpm-zsh/zpm) -  zsh的插件管理器类似于vim-plug.
* [zsh-autosuggestions](https://github.com/zsh-users/zsh-autosuggestions) -  zsh的鱼类自动建议.
* [zsh-pony](https://github.com/mika/zsh-pony) - 涵盖自定义ZSH而无需框架.
* [zsh-quickstart-kit](https://github.com/unixorn/zsh-quickstart-kit) - 使用zsh和zgen的简单zsh快速入门.
* [zsh-syntax-highlighting](https://github.com/zsh-users/zsh-syntax-highlighting) - 鱼壳像Zsh的语法高亮.
* [zshdb](https://github.com/rocky/zshdb) -  ZSH调试器.
* [ztanesh](https://github.com/miohtama/ztanesh) - 旨在通过ztanesh项目提供的配置提高UNIX命令行体验和工作效率.
* [zulu](https://github.com/zulu-zsh/zulu) -  ZSH 5+的总环境经理.
* [zunit](https://github.com/molovo/zunit) -  ZSH强大的单元测试框架.

### Bash

* [aliases](https://github.com/sebglazebrook/aliases) -  bash shell的上下文，动态，有组织的别名.
* [bash-it](https://github.com/Bash-it/bash-it) - 社区Bash命令和脚本的集合.

* [liquidprompt](https://github.com/nojhan/liquidprompt) - 针对Bash和Zsh的全功能和精心设计的自适应提示.

### Git
