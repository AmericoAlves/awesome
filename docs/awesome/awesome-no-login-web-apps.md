精选的令人敬畏的网络应用程序列表，无需您创建帐户即可使用. 

#### Inspiration

为了拯救世界，不为每个该死的东西创建用户帐户和安装软件应用程序.

#### Some pointers

 *此列表中链接的Web应用程序无需登录即可提供其大部分核心功能.  由于缺乏适当的替代方案，一些需要注册的网站也被列入此列表.  他们的列表中标有`[Account]`.
 *此处提到的大多数应用程序都提供某种用户帐户功能来保存历史记录和进度.  但如前所述，用户帐户不必使用这些应用程序的主要功能.
